1. Какие http запросы существуют?
POST, GET, delete, update и тд


2. Какие запросы может отдавать html файл?
Только POST и GET

3. Что чего нужен каждый запрос?
GET-запросы: используются для получения данных с сервера. Форма отображается 
пользователю.

POST-запросы: используются для отправки данных на сервер. Форма отправляется на сервер для 
обработки.

delete для удаления 
update для обновления
