1. С чем отличает метод get от filter?
Разница между методами get () и filter () заключается в том, 
что метод filter () возвращает набор запросов объектов, 
а метод get () возвращает один объект.

2. Как создается запрос по связям?
с помощью метода filter указывая какое то значение из связанной таблицы
Book.objects.filter(author__name="Dostoevsky")
